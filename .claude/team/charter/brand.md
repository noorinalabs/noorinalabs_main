# Brand Name

The display name is **Noorina Labs** (two words, with a space). All user-facing text — page titles, headings, meta tags, copy, documentation, error messages — MUST use "Noorina Labs."

Code identifiers remain `noorinalabs` (no space, lowercase): org name, repo names, package names (`@noorinalabs/*`), import paths, CSS classes, URLs, domain (`noorinalabs.com`).

| Context | Correct | Wrong |
|---------|---------|-------|
| Page heading | Noorina Labs | NoorinALabs |
| Meta og:site_name | Noorina Labs | NoorinALabs |
| npm scope | `@noorinalabs/design-system` | `@noorina-labs/design-system` |
| GitHub org | `noorinalabs` | `noorina-labs` |
